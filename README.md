# Idyllic Dream Helper
파이널판타지14 황금의 유산 확장팩의 아르카디아 선수권: 헤비급 4(영웅) 후반전의 '아르카디아의 꿈' 기믹 처리를 돕는 툴입니다.<br/>
버튼을 눌러 외워야 할 요소들을 기록하고 확인할 수 있습니다.

## 기능 및 사용법
*더블 모니터 사용을 가정하고 만들었습니다.* <br/>
*어떻게 오른쪽에 세로로 정렬하면 모니터 하나로도 사용할... 수 있긴 합니다.* <br/>
**첫 실행 시, 출력 패널이 가끔 열리지 않는 경우가 있습니다. 프로그램을 끄고 재실행 해 주세요.**

- 창이 2개로 나뉘며, 버튼을 누르는 패널과 출력패널을 따로 드래그하여 옮길 수 있습니다.<br/>
- 출력 패널은 항상 맨 위로 고정됩니다.
- '위치 잠금', '클릭 무시' 기능을 이용하여 출력패널을 조정할 수 있습니다.<br/>
- '맨 위 고정'을 체크하여 버튼을 누르는 창을 항상 맨 위로 고정할 수 있습니다.
- 처리법은 kr서버의 09stop, jp서버의 Game8 중 하나를 선택하여 사용할 수 있습니다.<br/>
- 슬라이더를 조작하여 투명도를 조절할 수 있습니다. 최소 40%까지 가능합니다.
### 출력 패널에서 표시하는 것
- 첫 기믹 체크(십자, X자 체크): 분신의 첫 생성이 십자인지, X자인지 표시합니다.
  - 메인조, 서브조에 따라서 동선을 표시합니다.
- 첫 안전지대(AC확인): 심상세계에서의 첫 안전지대(A 또는 C)를 표시합니다.
  - 예시) A일 경우 A쪽 양옆 안전지대.
- 교대: 심상세계의 타워에서 공격을 맞아 교대해야하는 상황인지 확인합니다. 기본값은 '교대 없음(X)' 입니다.
- 선 종류: 본인이 채야하는 선의 종류를 아이콘으로 표시합니다.
- 선 위치: 본인이 채야하는 선을 가지고 있는 분신의 위치 두 곳을 알려줍니다.
  - 09stop의 경우 부득이하게 바닥징으로 표시했습니다...
  - 예시 1) B, 2일 경우, 3시, 5시 분신을 의미합니다.
  - 예시 2) C, 3일 경우, 6시, 7시 분신을 의미합니다.
- 산개, 쉐어 동선: '12시 분신의 징(쉐어 or 산개)'과 본인의 '분신 위치'가 확인되면, 현실 세계에서의 4연속 쉐어, 산개를 처리할 때 본인이 가야하는 순서와 위치를 알려줍니다.<br/>
  - **정확한 위치가 아닌 방향만 대략적으로 알려주므로, 자세한 위치는 공략을 직접 확인해주세요.**
- 이동할 섬: '첫 안전지대'와 '필드에 남은 분신', '이동할 섬 확인'이 확인되면, 이동할 섬 방향와 어느쪽으로 서야할지 표시합니다.
  - 또한, 처리할 때 '히트박스 안' 혹은 '히트박스 바깥'을 표시합니다.
  - 예시) 양옆 안전이라면, 부채꼴 처리 시, 둘 다 보스의 히트박스 안으로 들어가면 안전합니다.
- 그 외: 심상세계 타워 처리 시, 피격당한 공격이나 디버프에 따른 기믹 처리 위치를 기록했습니다.
- 현재 처리법: 현재 사용하고 있는 처리법을 표시합니다.
